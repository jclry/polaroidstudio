# 需求文档

## 介绍
为拍立得相框添加纸张纹理，以增强视觉真实感。

## 需求

### 需求 1 - 纹理叠加
**用户故事：** 作为用户，我希望生成的拍立得照片相框具有纹理质感，而不是纯色背景，这样看起来更像真实的拍立得。

#### 验收标准
1. **When** 拍立得卡片（`.polaroid-card`）被渲染时，**the System shall** 在其背景色之上叠加 `assets/layout.svg` 纹理。
2. **When** 幽灵卡片（`.ghost-card`）在动画过程中出现时，**the System shall** 同样显示该纹理。
3. **The System shall** 确保纹理覆盖整个卡片区域（包括边框），且不遮挡照片内容（或作为背景层存在）。
